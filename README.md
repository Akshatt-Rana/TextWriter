# TextWriter
Animate your text like never before

## Usage

Add this in your root build.gradle at the end of repositories:

```bash
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
		}
	}
```
Add the dependency:

```bash
implementation 'com.github.sarnavakonar:TextWriter:v1.0'
```
